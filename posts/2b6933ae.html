学习路径

Author: zxffan Date: 九月 15, 2020   Category: Plan

知识概览

从基本功角度看

基本功扎实了，后面的知识技术才好推进。

工具

  1. terminal：命令行肯定要会用
  2. Git版本控制：原理、概念以及常见的场景，必须学会git的命令行使用方法。不能过分依赖GUI工具
  3. 编辑器：VSCode，这玩意还用学？错，你看过vscode的更新日志吗？你会用vscode来debug吗？你会写vscode插件吗？

开源常识

作为程序员，技术上紧跟开源社区准没错。

  1. 语义版本号
  2. GitHub、Gitee、Gitlab
  3. 开源许可

支撑知识

  1. 数据结构
  2. 设计模式
  3. 网络：了解HTTP吗？HTTP0.9 - 3.0有什么区别？HTTP的头信息有哪些？干什么用的？DNS？
  4. 浏览器工作模型

从语言角度看

前端开发四大语言：HTML CSS JavaScript WebAssembly

每一门语言都涉及到知识的延申

html5提供了繁多复杂的API

CSS 的预处理器有less、scss、stylus

JavaScript 的超集：TypeScript

WebAssembly需要学习一门系统级的语言——c++或者rust

从平台角度看

web技术的跨平台性实属最强。web技术几乎啥都能做。

网页端：没啥可说的，web最初就是用来干这个的

移动端：小程序、react native、flutter（也算上吧），还有一堆混合开发工具，PWA

客户端：electron

服务端：express、koa、nest.js

从视图层角度看

视图层有三大框架：Vue、React、Angular

Vue和React都是用组件化开发的思想来构建web应用。但其底层实现方式完全不一样。学习它们时一定要深入底层，比如搞清楚Vue响应式究竟是如何实现的。

无论Vue还是React都是所谓的**单页面应用(Single Page Application, SPA)。不利于SEO，首屏加载慢，进而又出现了服务端渲染(Server Side Render, SSR)**。

从工程化角度

前端工程化是基于nodejs的。

前端项目的构建工具有很多，常用的有Webpack、Rollup、Gulp/Grunt等，今年有Vue团队又出了个Vite。他们又有什么区别？

服务端渲染

微前端

工程化又要考虑哪些细节问题？（太多了）

node的包管理有npm和yarn两种，他们又有什么区别？

从服务端角度看

JavaScript可以通过nodejs来写服务端程序。不过又冒出了个deno，引入了十分有趣的概念，他们有什么区别呢？

至少掌握一个服务端的框架吧：轻量级的有Express、Koa。重量级的比较推荐Nest.js

有了服务端，是不是可以引入BFF层了

有了服务端，是不是可以做服务端渲染了

有了服务端，是不是可以用GraphQL了

有了服务端，是不是搭建Mock服务，性能监控、错误监控了

有了服务端，是不是还可以学习下数据库

阶段零：习惯养成

阶段零是贯穿整个学习计划的，旨在养成自律的学习习惯。

  • 每天至少一道算法题，LeetCode刷起来。（每道题都要写答题笔记哦）

阶段一：夯实基础

会从前端四大语言入手，HTML CSS JavaScript 三者结合起来学习。WebAssembly也会深入了解

  • JavaScript现代教程结合红宝书，搞懂JavaScript的方方面面。
  • 梳理H5新增特性，比如canvas、音视频方面（这需要结合JS学习）
  • 学习CSS基本概念，掌握页面布局以及众多的CSS3属性用法
  • TypeScript学习
  • WebAssembly的实现原理，手摸手写一下wasm

这个过程中也会穿插学习一些其它内容，比如浏览器工作模型等。

阶段二：视图层库

这个阶段会对React和Vue生态进行学习。没错。两个都要学！

  • 学习两者开发思路。掌握其提供每一个API
  • 尝试写个组件库
  • 深入源码，搞懂其设计思想

阅读源码过程中，会穿插一下设计模式以及数据结构的学习

阶段三：工程化

从工程化角度思考前端建设。

  • 构建工具学习
    • 学会webpack构建项目
    • 深入了解webpack原理
    • 了解其它构建工具rollup和gulp
  • 前端工程化的应用场景
  • 引入中间层BFF，研究一下前端体系建设。
  • 微前端的应用

阶段四：多端开发

这个阶段就涉及到其它场景了。比如说移动端、客户端甚至服务端。

  • 移动端
    • 熟练编写移动端页面，（移动端对交互有一定的要求）
    • 掌握 React native开发移动端应用
  • 客户端：Electron技术
  • 服务端：nodejs服务端通常用来搭建BFF层的。
    • 掌握Koa原理，学会开发中间件。拆一下Vite源码瞅一下
    • nest.js：如果真的要用node编写服务端，它是目前最好的选择
