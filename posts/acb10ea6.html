打包准备

  1. 证书
    1. mac: 开发者证书
    2. window：赛门铁克(Symantec)、WoSign
  2. 对于系统的机器（mac打macos的包，windows打win的包，linux打linux的包）
  3. 软件所需图片
    1. mac：
      1. 软件图标、icns格式，通过 image2icon 或者 iconutil 生成
      2. dmg 背景图
      3. 安装包图标
    2. Windows
      1. ico
      2. installerIcon-安装图标
      3. unInstallerIcon-卸载图标

使用electron-builder来打包

安装相关依赖

  • npm i -g --production window-build-tool（用管理员权限安装，window必备）
  • 项目目录下执行npm i electron-builder -S
  • 在package.json中，添加"postinstall": "electron-builder install-app-deps"。每当使用npm install或者yarn add时，会自动安装匹配当前electron版本的依赖。

总结

  1. 发布产品注意版本号的升级

    1. npm version patch：自动升级patch
    2. npm version minor：自动升级minor，path会清0
    3. npm version major：自动升级major，minor和patch会清0

  2. 证书

    1. windows下没有证书，可能会被杀软误杀
    2. mac下没有证书，无法实现自动更新
    3. windows下打包可以通过修改nsis来自定义安装逻辑
    4. 开源软件可以基于 Travis, AppVeyor 持续集成
