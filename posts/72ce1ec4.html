Electron-集成原生能力

Author: zxfan Date: 八月 2, 2020   Category: Electron

NAPI

N-API是nodejs的一部分，独立于v8 runtime，用来在nodejs中集成C++：同一ABI（Application Binary Interface）、无需重新编译

  • 本身是基于C的API
  • C++封装 node-addon-api

环境配置

yarn global add windows-build-tools : 管理员身份安装

yarn global add node-gyp: gyp即generate your project

初始化项目

  1. 初始化：
    1. npm init -y
    2. yarn add bindings node-addon-api -D
    3. package.json增加"gyp":true
  2. 目录结构

binding.gyp

bindings用来加载原生模块(.node文件)。不同平台不同版本rebuild出来的目录结构可能不一样。通过bindings来抹平差异。

使用

通常做法是，将写好的node addon发布到npm上。在其它项目中就可以通过npm或者yarn安装，然后就require使用

集成DLL（动态链接库）

node-ffi-napi，黑盒调用

常见错误：

Dynamic Linking Error: Win32 error 126

  • dll 路径没写对、arch 没选对、dll 引用有问题

  • 传参有问题、dll 没有这个函数
